# Корзина интернет-магазина

Консольное приложение для работы с корзиной покупок.

## Функционал

- Добавление товаров в корзину
- Удаление товаров из корзины по названию
- Просмотр содержимого корзины
- Расчет общей суммы покупок

## Структура проекта

- `Product.java` - класс товара
- `Cart.java` - класс корзины
- `Main.java` - основной класс с тестовыми примерами

## Требования

- Java 17 или выше
- Любая среда разработки (IntelliJ IDEA, Eclipse, VS Code) или командная строка

## Запуск приложения

### Способ 1: В среде разработки

1. Откройте проект в вашей IDE
2. Скомпилируйте все три файла
3. Запустите класс `Main`

### Способ 2: Из командной строки

```bash
# Компиляция всех файлов
javac Product.java Cart.java Main.java

# Запуск приложения
java Main
